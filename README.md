## Description

HoneyBeeLoveless is a simple discord honeypot channel bot. It will softban anyone who types a message into the monitored channel. 

You will need to give the bot privileged Discord Intents to use this bot
- Guilds
- Guild Members
- Message Content

## Usage

How to use HoneyBeeLoveless
1. Clone this repository
2. Copy `config.json.example` to `config.json`
3. Set your bot token and monitored discord channel IDs in `config.json`
4. Run the bot using NodeJS (`npm start` also works)
